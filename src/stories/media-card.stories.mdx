### Card markup structure

```html
<div className="card-list__item">
  <div className="card-list-item__image">
    <img src="" alt="" />
  </div>
  <div className="card-list-item__content">
    <h1 className="card-list-item__title">Sir Nicholas</h1>
    <div className="card-list-item__details">...</div>
  </div>
</div>
```

To make an entire card clickable, wrap it in an anchor with a `card-list-item__wrapper-link` class:

```html
<a href="" class="card-list-item__wrapper-link">
  <div class="card-list__item">...</div>
</a>
```

To display multiple cards, wrap them in a `card-list` container:

```html
<section class="card-list">
  <article class="card-list__item">...</article>
  <article class="card-list__item">...</article>
</section>
```
