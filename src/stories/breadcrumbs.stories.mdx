| Breadcrumbs elements |    |
| ---- | ---- |
| base element | <code>.breadcrumbs</code> |
| layout container (provides flex container with height and max width) | <code>.breadcrumbs__layout</code> |
| page link | <code>.breadcrumbs__page</code> |
| active page class (appended) | <code>.breadcrumbs__current-page</code> |
| link text | <code>.text</code> |
| link icon | <code>.icon</code> (link icon is only shown on mobile screens, and replaces the text link) |
| divider icon | <code>.breadcrumbs__divider-icon</code> |

### Usage
```html
<nav aria-label="Breadcrumb" className="breadcrumbs">
  <div className="breadcrumbs__layout">
    <a className="breadcrumbs__page" aria-label="Parent Page Name" href="">

    <div className="breadcrumbs__page breadcrumbs__current-page" aria-current="location">
```
