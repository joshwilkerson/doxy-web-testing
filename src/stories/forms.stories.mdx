| Number Input classes |  |
| ---- | ---- |
| Wrapper | <code>.number-spinner</code> | 
| Input | <code>.number-spinner__input</code> | 
| Buttons | <code>.number-spinner__btn</code> | 
| Size | <code>.sm-number-spinner</code> |

### Usage
*Note: you'll have to hook up your own js to make the buttons increment and decrement, set your step, min, and max values, etc.*
```html
<div className="number-spinner">
  <button className="number-spinner__btn" aria-label="decrement input value">-</button>
  <input 
    type="text" 
    className="number-spinner__input" 
    name="the-numbers" 
    id="the-numbers" 
    placeholder="0" 
    min="1" 
    max="24" 
    step="1" 
  />
  <button className="number-spinner__btn" aria-label="increment input value">+</button>
</div>
```
