<!-- prettier-ignore -->
| Modal elements |  |
| ---- | ---- |
| base element | <code>.modal</code> |
| header | <code>.modal__header</code> |
| title | <code>.modal-title</code> |
| body | <code>.modal__body</code> |

<!-- prettier-ignore -->
| Modal control |
| ---- |
| modal is hidden by default; will be visible when a parent class name of <code>.modal--open</code> is present. |
| <code>.modal-layer</code> should wrap <code>.modal</code> in order for the modal to have the appearance of sitting above the page. |

### Modal markup structure

```html
<body class="modal--open">
    <div class="modal-layer">
        <div className="modal">
            <div className="modal__header">
                <div className="modal-title">...</div>
            </div>
            <div className="modal__body">
                ...
            </div>
            <div className="modal__footer">
                ...
            </div>
        </div>
    </div>
</body>
```
