## Badge

To differentiate badges from buttons, badges should be tighter, lighter, and less rounded.

export function SampleBadgeLayout(props) {
  return (
    <div style={{
      alignItems: "center",
      display: "flex",
      gap: "0.5rem",
    }} {...props} />
  );
}
