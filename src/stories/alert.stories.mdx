| Alert classes |  |
| ---- | ---- |
| Base | <code>.alert</code> | 
| Options (appended) | <code>.info-alert</code> <code>.success-alert</code> <code>.warning-alert</code> <code>.error-alert</code> |
| Variation (appended) | <code>.quiet-alert</code> <code>.loud-alert</code> |
| Size (appended) | <code>.sm-alert</code> |

### Usage
```html
<div class="quiet-alert success-alert alert">A quiet success alert.</div>
```
